Difference Between map, foreach, filter and find

.forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.
.map(), executes the same code on every element in an array and returns a new array with the updated elements.
.filter(), checks every element in an array to see if it meets conditions and returns a new array with All the element that fullfilled the conditions. .find(), it check every elements in an array to see it fullfill the conditions and return one one element among the matched items.

Difference Between Arrow and Regular Function

Unlike regular functions, arrow functions do not have their own this
Arguments objects are not available in arrow functions, but are available in Regular functions.
Regular functions created using function declarations or expressions are ‘constructible’ and ‘callable’. Since regular functions are constructible, they can be called using the ‘new’ keyword. However, the arrow functions are only ‘callable’ and not constructible. Thus, we will get a run-time error on trying to construct a non-constructible arrow functions using the new keyword.

Difference Between Var, Let and Const

var variable scope is functional scope.It can be updated and re-declared into the scope.
let let variable scope is block scope. It can be update but can't redeclare in the scope. It can be declare without initialization.
const The const variable can't be declare without initialization and can't access without initialization. In the const variable value can't update or modify, its fixed.

Why We Use Template String?

Template string is enclosed by backtick (``) characters instead of quotes.
Along with having normal strings, template literals can also contain other parts called placeholders, using ${expression}. The strings and placeholders get passed to a function — either a default function, or a function we supply.
When We need to write multiline string, we use teplate string. Also we can use ternary operator using template string without nesting.
